# Technologies

---

## Node.js

[Node.js](https://nodejs.org) is a JavaScript runtime built on Chrome's V8 engine. In Start UI it powers the development tooling and server-side runtime (scripts, server functions, and more).

[GitHub](https://github.com/nodejs/node) · [License MIT](https://github.com/nodejs/node/blob/main/LICENSE)

---

## TypeScript

JavaScript is an untyped language. [TypeScript](https://www.typescriptlang.org/) adds static types to help define contracts between functions and components, improving reliability and developer experience across the codebase. Adoption is incremental and optional, but recommended.

[GitHub](https://github.com/microsoft/TypeScript) · [License Apache 2.0](https://github.com/microsoft/TypeScript/blob/main/LICENSE.txt)

---

## React

[React](https://react.dev/) is a JavaScript library for building reactive user interfaces. It has a large ecosystem and community, which helps long-term maintainability and availability of resources.

[GitHub](https://github.com/facebook/react) · [License MIT](https://github.com/facebook/react/blob/main/LICENSE)

---

## TanStack Start

[TanStack Start](https://tanstack.com/start) is a full‑stack React framework powered by TanStack Router. It provides full‑document SSR, streaming, server functions/RPCs, bundling via Vite, and deployment to virtually any runtime.

[GitHub](https://github.com/TanStack/router) · [License MIT](https://github.com/TanStack/router/blob/main/LICENSE)

---

## Tailwind CSS

[Tailwind CSS](https://tailwindcss.com/) is a utility‑first CSS framework that enables rapid UI development with composable classes and an extensible design system.

[GitHub](https://github.com/tailwindlabs/tailwindcss) · [License MIT](https://github.com/tailwindlabs/tailwindcss/blob/master/LICENSE)

---

## shadcn/ui

[shadcn/ui](https://ui.shadcn.com/) is a collection of copy‑and‑paste React components built with Tailwind CSS and Radix UI primitives. It provides accessible, unstyled components you can fully customize.

[GitHub](https://github.com/shadcn-ui/ui) · [License MIT](https://github.com/shadcn-ui/ui/blob/main/LICENSE.md)

---

## React Hook Form

To create forms in React, [React Hook Form](https://react-hook-form.com/) offers performant, flexible, and extensible form handling with easy‑to‑use validation.

[GitHub](https://github.com/react-hook-form/react-hook-form) · [License MIT](https://github.com/react-hook-form/react-hook-form/blob/main/LICENSE)

---

## oRPC

[oRPC](https://orpc.unnoq.com/) combines RPC with OpenAPI to build end‑to‑end typesafe APIs. It supports contract‑first development, multiple schema validators (Zod, Valibot, ArkType), streaming/SSE, and integrations with TanStack Query and more.

[GitHub](https://github.com/unnoq/orpc) · [License MIT](https://github.com/unnoq/orpc/blob/main/LICENSE)

---

## Prisma

[Prisma](https://www.prisma.io/) is an open‑source database toolkit and ORM for Node.js and TypeScript that simplifies database access, migrations, and data modeling.

[GitHub](https://github.com/prisma/prisma) · [License Apache 2.0](https://github.com/prisma/prisma/blob/main/LICENSE)

---

## Better Auth

[Better Auth](https://www.better-auth.com/) is a framework‑agnostic authentication and authorization library for TypeScript with a rich plugin ecosystem (OAuth, 2FA, multi‑tenant, and more).

[GitHub](https://github.com/better-auth/better-auth) · [License MIT](https://github.com/better-auth/better-auth/blob/canary/LICENSE.md)

---

## Storybook

[Storybook](https://storybook.js.org/) is an open‑source tool to develop UI components in isolation, document them, and build a shared component library.

[GitHub](https://github.com/storybookjs/storybook) · [License MIT](https://github.com/storybookjs/storybook/blob/next/LICENSE)

---

## Vitest

[Vitest](https://vitest.dev/) is a fast unit test framework built for Vite projects, offering instant ESM support, TypeScript integration, and great DX.

[GitHub](https://github.com/vitest-dev/vitest) · [License MIT](https://github.com/vitest-dev/vitest/blob/main/LICENSE)

---

## Playwright

[Playwright](https://playwright.dev/) is a modern end‑to‑end testing framework for web applications with cross‑browser automation and powerful tooling. Start UI end‑to‑end tests are written using Playwright.

[GitHub](https://github.com/microsoft/playwright) · [License Apache 2.0](https://github.com/microsoft/playwright/blob/main/LICENSE)
